# Milestone — First successful WrathOS ISO boot

Date: March 2026

WrathOS 0.1 alpha successfully booted from ISO with:
- Base: Debian GNU/Linux 13 (trixie)
- Kernel: 6.19.9-cachy (CachyOS kernel confirmed)
- Audio: PipeWire 1.4.2
- Graphics: Mesa/LLVM (llvmpipe in VM, real GPU on hardware)
- Display: KDE Plasma
