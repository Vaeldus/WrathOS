WrathOS APT Repository

To add this repository:

curl -fsSL https://vaeldus.github.io/WrathOS/apt/wrathos-archive-keyring.gpg | sudo tee /etc/apt/trusted.gpg.d/wrathos.gpg
echo "deb [signed-by=/etc/apt/trusted.gpg.d/wrathos.gpg] https://vaeldus.github.io/WrathOS/apt trixie main" | sudo tee /etc/apt/sources.list.d/wrathos.list
sudo apt-get update
